User Management

Viewing Users

Navigate to the "Users" tab to see all registered users. You can:

  • Search users by name or email
  • Filter by plan tier (Free, Pro, Team)
  • View usage statistics
  • Export user list to CSV

Gallery Management

Featured Apps

Featured apps appear prominently in the Community Gallery. To feature an app:

  1. Navigate to "App Gallery" tab
  2. Find the app you want to feature
  3. Click the star icon or "Feature" button
  4. The app will be marked as featured

Moderation Guidelines

When reviewing apps for the gallery, check for:

  • Original content (not copied)
  • Functional code (no broken apps)
  • Appropriate content (no offensive material)
  • Quality design and usability

Data Storage

LocalStorage Structure

AppForge uses browser LocalStorage for data persistence. Key data stores:

// User data
localStorage.getItem('appforge_user')
localStorage.getItem('appforge_settings')
localStorage.getItem('appforge_ai_settings')

// App data
localStorage.getItem('appforge_currentCode')
localStorage.getItem('appforge_gallery')

// Gamification
localStorage.getItem('appforge_achievements')
localStorage.getItem('appforge_gamification')

Backup Strategy

Current data is stored client-side. For production, implement:

  • Firebase Realtime Database for user data
  • Firebase Storage for app backups
  • Regular automated backups
  • Export functionality for users

Troubleshooting

Common Issues

AI Not Responding

If Puter.js AI is not responding:

  1. Check internet connection
  2. Verify Puter.js script is loaded
  3. Try switching to a different AI provider
  4. Clear browser cache and reload

Export Not Working

If exports are failing:

  1. Check if user has reached export limit
  2. Verify code is not empty
  3. Check browser console for errors
